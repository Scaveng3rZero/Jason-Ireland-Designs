Featured Project

Typographic Portrait

A text-based portrait design built through typographic arrangement and repetition. This piece explores how type can function as both language and image, using lyric fragments and varied scale to construct a recognizable visual form.

Typography / Composition

Typographic Portrait Study

This project uses typography as both content and structure, arranging text at different weights, sizes, and densities to build a portrait through contrast, spacing, and shape recognition. Rather than relying on traditional illustration, the image is formed through layout decisions and typographic hierarchy.

The piece reflects an interest in experimental composition and demonstrates how graphic design can communicate visually even when built from nothing but type. It also shows an understanding of rhythm, negative space, and balance within a highly detailed composition.

Design Focus

What This Project Demonstrates

Typographic Image-Making

This work shows how typography can be used beyond readable text to create image, texture, and form within a composition.

Composition & Contrast

Scale shifts, density changes, and black-and-white contrast guide the eye and help define the portrait while maintaining visual cohesion.

Experimental Design Thinking

The project reflects a willingness to push design beyond standard layouts and explore more conceptual approaches to communication.
